Deskripsi

Aplikasi ini dirancang untuk mengelola pembayaran sekolah, cicilan, dan administrasi keuangan siswa. Dengan menggunakan Laravel Livewire, sistem ini menawarkan interaksi cepat tanpa reload halaman.

Dashboard admin dikembangkan menggunakan Filament PHP untuk mempermudah pengelolaan transaksi dan laporan keuangan sekolah. Selain itu, sistem pembayaran diintegrasikan dengan Midtrans untuk mendukung pembayaran online melalui QRIS, Virtual Account, dan e-Wallet.
