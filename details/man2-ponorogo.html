Description

This application is designed to manage school payments, installments, and student financial administration. Using Laravel Livewire, the system offers quick interactions without page reloads.

The admin dashboard is developed using Filament PHP to simplify school financial transaction and report management. Additionally, the payment system is integrated with Midtrans to support online payments through QRIS, Virtual Account, and e-Wallet.
