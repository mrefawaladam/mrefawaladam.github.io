Description

rSchoolToday is a cloud-based school management platform providing scheduling, communication, and school administration features. As a Backend Developer in the Scheduling division, I contributed to developing core features and internal tools to optimize the school scheduling system.

This project utilizes Symfony as the backend framework with frontend development using Twig, Vue.js, and Next.js for modern and responsive interfaces.

Responsibilities & Contributions

  • 🔹 Backend Development with Symfony – Building APIs, optimizing databases with Doctrine ORM, and implementing role-based access control.
  • 🔹 Scheduling Feature Development – Designing automated scheduling systems and event management for classes, exams, and school activities.
  • 🔹 Internal Tools Development – Developing monitoring systems, debugging tools, and admin dashboards for event and schedule management.
  • 🔹 Frontend Integration – Using Twig for template rendering and developing Vue.js / Next.js components for interactive UI.

Key Features

  • 📅 Smart Scheduling – Automated algorithms for class & event scheduling.
  • 📢 Real-time Notifications – Email & push notifications for schedule changes.
  • 🔗 Calendar Integration – Synchronization with Google Calendar & Outlook.
  • 🖥️ Admin Dashboard – School schedule & event management in one platform.
  • 👩‍🏫 Role-based Access – Role-based access system for admins, teachers, & students.

Experience & Learning

  • ✅ Implementing Symfony in large-scale systems.
  • ✅ Developing scheduling algorithms & event-based systems.
  • ✅ Optimizing backend performance for high-traffic applications.
  • ✅ Collaborating with frontend teams using Vue.js & Next.js.

Through this project, I gained experience in building complex scheduling systems, improving school administration efficiency, and ensuring more structured & automated schedule management. 🎓📆🚀
